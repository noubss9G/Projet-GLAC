Approche de développement

SB Admin 2 utilise largement les classes utilitaires de Bootstrap 4 afin de réduire l'encombrement du CSS et les mauvaises performances des pages. Des classes CSS personnalisées sont utilisées pour créer des composants personnalisés et des classes utilitaires personnalisées.

Avant de travailler avec ce thème, vous devez vous familiariser avec le framework Bootstrap, en particulier les classes utilitaires.
